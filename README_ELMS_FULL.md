## 1. Introduction

The **E-Learning Management System (ELMS)** is a web-based application that allows students to enroll in courses, access learning materials, submit assignments, and take quizzes. Instructors manage courses, while administrators control the overall system.

## 2. Problem Statement

Manual management of courses and student records is time-consuming. ELMS provides one centralized platform to manage all academic activities.

## 3. Objectives

- Secure Login
- Course Management
- Student Enrollment
- Assignment & Quiz Management
- Database Management

## 4. Scope and Modules

### Authentication
- Registration
- Login
- Role-Based Access

### Course Management
- Create, Update & Delete Courses
- Upload Study Materials

### Student Module
- Enroll in Courses
- View Materials
- Submit Assignments
- Take Quizzes
- View Progress

### Admin Module
- Manage Users
- Manage Courses
- View Reports
